#knockout-firebase
Knockout tools for using Firebase as its DB

#Goals
 1. Sync just a value
 2. Sync an object i.e., a collection of values
 3. Sync an array
 4. Sync a combination of 1-3
 5. Sync can be read-only or bi-directional
